## UI Components

- **Form Container**: Centered card with max-width 500px
- **Input Fields**: Full-width text inputs with labels
- **Password Toggle**: Eye icon to show/hide password
- **Validation Messages**: Red text with warning icon below invalid fields
- **Submit Button**: Primary CTA button, full-width, disabled during loading
- **Loading State**: Spinner with "Creating account..." text
- **Success Toast**: Green banner at top of dashboard after redirect
- **Link to Login**: Text link below form for existing users

## Accessibility

- All inputs have associated labels
- Error messages linked to inputs via aria-describedby
- Submit button disabled during submission
- Focus management after errors
- Keyboard navigation supported
- Screen reader announcements for validation errors
