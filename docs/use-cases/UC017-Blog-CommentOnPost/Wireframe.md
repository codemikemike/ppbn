# UC017 — Wireframe

## Blog Post Page (`/blog/[slug]`)

- Post header + details (existing)
- Post content (existing)
- Comments section (new)
  - Heading: "Comments"
  - Comment list (approved only)
    - For each comment:
      - Author name
      - Date (<time>)
      - Content
  - Comment form (new)
    - Textarea: comment content (min 5 chars)
    - Submit button
    - Logged out state: "Login to comment"
    - Success/error feedback
